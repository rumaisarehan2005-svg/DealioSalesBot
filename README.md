[![Hugging Face](https://img.shields.io/badge/Hugging%20Face-AI-FFD21E?logo=huggingface&logoColor=black)](https://huggingface.co)

**Dealio** is a powerful Discord sales bot designed specifically for Twitch streamers. It sells avatars, overlays, emotes, and streaming assets with a complete shopping cart system and AI-powered sales assistant.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🛒 **Shopping Cart** | Add/remove products, view cart, complete checkout |
| 🤖 **AI Sales Assistant** | Answers questions using Hugging Face AI (Meta Llama 3.1) |
| 📦 **Product Catalog** | Avatars, Overlays, Emotes, Stream Assets |
| 💰 **Order Management** | Tracks orders with unique IDs |
| 📧 **DM Payment Instructions** | Sends payment details via DM |
| 🔥 **Deals & Discounts** | Built-in promo code system |
| 🎫 **Support System** | Customer support integration |

## 🚀 Commands

### 🛒 Shopping Commands

```bash
!shop           # Show all categories
!products       # List all products with prices
!category <name>  # Show products in category (avatars/overlays/emotes/assets)
!price <category> <product>  # Check product price
!preview <category> <product> # Preview product
